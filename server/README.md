# PrimeTrade Dashboard API

> Backend server for the PrimeTrade Frontend Developer assignment.
> Provides JWT-based authentication, user profile management, and CRUD APIs for tasks.

---

## 🚀 Tech Stack

- Node.js
- Express
- TypeScript
- PostgreSQL
- Prisma ORM
- JWT Authentication
- bcrypt (password hashing)
- Zod (validation)
